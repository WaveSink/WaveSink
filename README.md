WaveSink is a lightweight and elegant audio routing utility for Windows. It allows you to seamlessly route system audio to multiple output devices simultaneously while providing custom, per-device volume and equalizer controls.

## Features

* **Multi-Device Playback:** Route your system audio to multiple output devices (sinks) at the same time.
* **Independent Equalizer:** Apply custom 5-band equalizer settings for each individual audio device.
* **Volume Control:** Manage the volume levels of each audio sink independently.
* **Profile Memory:** Automatically remembers your volume and equalizer configurations for each device.
* **Native UI:** Clean, responsive, and native Windows user interface that adapts to your system theme.

## Installation

1. Go to the **Releases** tab of this repository.
2. Download the latest `WaveSink_Installer.exe` setup file.
3. Run the installer and follow the on-screen instructions.
4. Launch WaveSink from your Start menu or desktop shortcut.
